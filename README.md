# Features

<h1 align="center">
  <img src="static/vscan-run.png" alt="vscan" width="850px"></a>
  <br>
</h1>

- 快速的端口扫、指纹探测功能
- 快速的登录密码爆破功能
- 快速的POC检测功能
- 轻量、开源、跨平台使用
- 支持多种类型的输入 - **STDIN/HOST/IP/CIDR/DOMAIN**
- 支持多种类型的导出 - **JSON/TXT/CSV/STDOUT**
